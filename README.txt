프린터 서버 - 최적화 독립 실행 버전
==================================================

특징:
✓ Python 설치 불필요
✓ 빠른 시작 (폴더 방식)
✓ 스플래시 스크린 표시
✓ 불필요한 패키지 제외로 최적화

실행 방법:
1. '프린터서버.exe'를 더블클릭
2. 스플래시 화면 후 GUI 시작
3. 프린터 선택 후 '시작' 버튼 클릭

시스템 요구사항:
- Windows 10/11
- 프린터 드라이버 설치됨

배포 방법:
- 폴더 전체를 복사하여 배포
- 단일 EXE 방식보다 5-10배 빠른 시작

API 엔드포인트:
- GET  /printers     : 프린터 목록
- POST /print-image  : 이미지 인쇄
- 문서: http://localhost:8000/docs
